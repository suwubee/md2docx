# Word模板文件夹

## 📁 用途

将您的 `.dotx` Word模板文件放在此文件夹中，程序会自动识别。

## 📝 使用方法

### 1. 添加模板

将模板文件（扩展名为 `.dotx`）复制到此文件夹：

```
templates_docx/
├── 公司报告模板.dotx
├── 技术方案模板.dotx
└── README.md
```

### 2. 程序自动识别

启动应用后，下拉菜单会自动显示所有可用模板。

### 3. 应用模板

转换时选择模板，生成的DOCX文档会自动应用模板中的样式。

## 🎨 创建模板

### 方法1: 修改现有文档

1. 打开Word，创建新文档
2. 设置样式：
   - 修改"标题1"、"标题2"等样式
   - 修改"正文"样式
   - 设置页边距、页眉页脚
3. **文件** → **另存为** → 选择 **Word模板 (*.dotx)**
4. 保存到此文件夹

### 方法2: 从现有文档提取

1. 打开已有的格式化文档
2. 删除所有内容（保留样式）
3. 另存为 `.dotx` 格式
4. 保存到此文件夹

## 📊 模板包含什么

### ✅ 会应用的内容

- 标题样式（字体、大小、颜色）
- 正文样式
- 表格样式
- 页面布局（页边距、纸张大小）
- 主题颜色
- 默认字体

### ❌ 不会包含的内容

- 模板文档中的具体文字
- 模板中的图片
- 页眉页脚的具体内容

## 💡 推荐设置

### 标题样式建议

```
标题1: 微软雅黑，24pt，加粗，蓝色
标题2: 微软雅黑，18pt，加粗，深蓝色
标题3: 微软雅黑，14pt，加粗
正文:   宋体，12pt，行距1.5倍
```

### 页面设置建议

```
页边距: 上下 2.54cm，左右 3.17cm
纸张:   A4
页眉:   1.5cm
页脚:   1.75cm
```

## 🔍 故障排查

### 问题：模板不显示

**解决：**
- 确认文件扩展名为 `.dotx`
- 确认文件在 `templates_docx` 文件夹中
- 重启应用

### 问题：样式没有应用

**解决：**
1. 在Word中打开模板
2. 检查"样式"面板，确认样式已自定义
3. 确保修改了"标题1"等样式，而不是直接格式化文字

### 问题：中文显示不正常

**解决：**
- 在模板中明确设置中文字体（宋体、微软雅黑等）
